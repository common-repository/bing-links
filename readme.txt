Bing Search Results/Links provides the top results for a combined search on your post tags using Bing PHP Library

== Description ==

This plugin provides a widget with the top Bing Search Results for a combined search query of your post tags using
the Bing PHP SDK.

== Installation ==

1. Upload the zip file contents to wp-content/plugins
2. Activate the plugin through the Plugins menu in WordPress
3. Set the widget location and potential options -- dynamic sidebar is required in your templateplates

== Frequently Asked Questions ==

= Can I set the number of results? =

Yes. Go to Settings, Appearance, Widgets and specify the number of results.
